oxis-style evangelion visualizer

python opencv computer vision evangelion

real-time computer vision overlays on evangelion footage, inspired by oxis's visual style. uses shi-tomasi corner detection, lucas-kanade optical flow tracking, and canny edge detection to generate tracking lines, coordinate readouts, and nerv-themed hud overlays. paired with the cruel angel's thesis.

the oxis technique: detect feature points in each frame, track them across frames with optical flow, then draw crosshairs, coordinate labels, motion trails, and connection lines between nearby points. all in the eva unit-01 color palette.
